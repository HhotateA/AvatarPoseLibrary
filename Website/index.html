AvatarPoseLibrary
  • 🔗 Github

AvatarPoseLibrary (APL) は、Unity 上で VRChat 向けのアバター用ポーズを管理・適用するためのツールです。アバターのポーズを効率的に制御し、VRChat アバターの制作やカスタマイズを支援します。

VCCに追加 UnityPackageをダウンロード チュートリアル

✨ 複数のメニュー統合

アニメーションを一元管理し、効率的な制御を実現します。

📸 サムネイル撮影機能

自動でサムネイルを撮影し、メニュー画像を差し替え可能です。

🎯 トラッキング抑制

指だけを動かす、無効化など柔軟に設定できます。

🔧 コンポーネントで設定完了

コンポーネントベースで、プレハブ化にも対応。

🧠 より人間らしい動きに

静的ポーズでも自然な動作に変換します。

📂 許可なしで再配布OK

アニメーションやポーズデータを自由に配布可能。
