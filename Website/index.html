AvatarPoseLibrary
  • 🔗 Github

AvatarPoseLibrary (APL) は、Unity 上で VRChat 向けのアバター用ポーズを管理・適用するためのツールです。アバターのポーズを効率的に制御し、VRChat アバターの制作やカスタマイズを支援します。

VCCに追加 UnityPackageをダウンロード マニュアル

✨ パラメーター最適化

複数のメニューを自動で統合し、アバターのパラメーター数を最小限にします！

📸 サムネイル撮影機能

自動でポーズのサムネイルを撮影し、メニューをアバターごとに差し替えます！

🎯 トラッキング制御

細やかにトラッキングを設定でき、指だけを動かすなどポーズごとに柔軟に変更できます！

🔧 コンポーネントで設定完了

コンポーネントベースで設定を行え、プレハブ化にも対応しています！

🧠 より人間らしい動きに変換

動きのないポーズを、自動で動画撮影に最適化された自然な動きに変換します！

📂 許諾無しで商用利用OK

本ツールに対応したアニメーションやポーズデータを、商用非商用問わず自由に配布できます！
