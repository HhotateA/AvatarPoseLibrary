# AvatarPoseLibrary Audio Pitch Sync Patch

Studio KEEP의 VRChat 영상 제작 및 퍼포먼스 workflow를 위해 제작한 AvatarPoseLibrary 확장 패치입니다.

An extension patch for AvatarPoseLibrary created for Studio KEEP's VRChat cinematic and performance production workflow.

---

# ✨ 기능 / Features

* 포즈별 AudioClip 설정 지원
  Per-pose AudioClip support

* Motion Speed와 오디오 재생 속도 실시간 동기화
  Realtime Motion Speed ↔ Audio playback speed synchronization

* AudioPitchLayer 추가
  Added AudioPitchLayer support

* VCC / VPM Repository 지원
  Added VCC / VPM repository support

* VRChat 퍼포먼스 및 뮤직비디오 제작 최적화
  Optimized for VRChat performances and cinematic/music video production

---

# 📦 설치 방법 / Installation

## VCC Repository 추가 / Add Repository to VCC

### 원클릭 추가 / One-click Add

```text id="rn86mj"
vcc://vpm/addRepo?url=https://3minn.github.io/AvatarPoseLibrary/packages.json
```

### 수동 추가 / Manual Repository URL

```text id="j2v1n5"
https://3minn.github.io/AvatarPoseLibrary/packages.json
```

VCC → Settings → Packages → Add Repository

# ⚠️ 주의사항 / Notes

* 본 패치는 AvatarPoseLibrary 기반 비공식 확장 패치입니다.
  This is an unofficial extension patch for AvatarPoseLibrary.

* 아바타 FX Animator 구조에 따라 일부 환경에서는 충돌 가능성이 있습니다.
  Depending on avatar FX Animator structure, conflicts may occur in some environments.

* Audio Pitch는 Animator 기반으로 동작하므로 급격한 속도 변화 시 단계식 변화가 발생할 수 있습니다.
  Audio Pitch synchronization is Animator-based and may show stepped transitions during rapid speed changes.
